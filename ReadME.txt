HOW TO RUN THE SYSTEM
Requirements;
**Download and Install any local web server such as XAMPP/WAMP.
**Download the provided source code zip file. (download button is located below)
**Installation/Setup

**Open your XAMPP/WAMP's Control Panel and start the "Apache" and "MySQL".
**Extract the downloaded source code file.
**If you are using XAMPP, copy the extracted source code folder and paste it into the XAMPP's "htdocs" directory. And If you are using WAMP, paste it into the "www" directory.
**Browse the PHPMyAdmin in a browser. i.e. http://localhost/phpmyadmin
**Create a new database naming "studentinfo".
**Import the provided SQL file. The file is known as "student.sql" located inside the extracted source code folder.
**Browse the Simple Student Information System in a browser. i.e. http://localhost/Basic_StudentInformationSystem.

MODULES EXPLANATION
**Student Information System Page (Index Page):- This is the first page(Landing Page) of the system and it display the
information of the students added to the system and other system features or modules. 
**Add/Create Page:- This Page allows user or student to create profile as a new student on the system.
**View/Read:- This is the page that shows or display registered students information on the system.
**Update Page:- This page allows existing user or students to update or change their information or details on the system. 
**Delete Module:- This allows existing user or student profile to be deleted from the system.
